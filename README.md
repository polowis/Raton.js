# Raton.js

![npm](https://img.shields.io/npm/v/raton.js.svg)

## Installation


```sh
$ npm install raton.js --save
```

### How to use

```javascript
const raton  = require('raton.js')

const client = new raton.Population()

client.nation.getAllPopulation()

```
#### Simple Request

```javascript

client.state.getPopulationByName('Georgia')

```
